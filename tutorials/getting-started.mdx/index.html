Getting Started with NoWires.AI

Welcome to NoWires.AI — the virtual breadboard that connects to real hardware. This tutorial walks you through signing in, navigating the interface, and creating your first project.

What you’ll learn

  • How to sign up and sign in
  • The main areas of the interface
  • What each sidebar icon does
  • How to create and save your first project

1. Sign up or sign in

Head to nowires.ai/app. If this is your first visit, click Sign Up and create an account with your email. If you already have an account, sign in with your credentials.

💡 Tip: Your projects are saved to the cloud, so you can access them from any device once you’re signed in.

2. First look at the interface

Once you’re in, you’ll see three main areas:

  • Sidebar (left edge) — icon buttons for navigating between panels
  • Board canvas (center) — the virtual breadboard where you build circuits
  • Toolbar (top of the canvas) — tools for selecting, wiring, erasing, zooming, and switching modes

The board itself is a realistic breadboard with labeled columns (1–63), rows (A–J), and color-coded power rails along the top and bottom edges.

3. The sidebar icons

The sidebar gives you quick access to every major feature. From top to bottom:

IconPanelWhat it does
PartsParts drawerBrowse and place electronic components
NetsNets drawerView and manage all electrical connections
BoardsBoards drawerRegister and manage physical NoWires hardware
ProjectsProject OrganizerCreate, open, duplicate, and delete projects
CommunityCommunity browserBrowse, fork, and share public projects
SettingsSettings panelTheme, preferences, and account options

Click any icon to open its panel. Click it again (or press Esc) to close it.

4. Create your first project

  1. Click the Projects icon in the sidebar to open the Project Organizer.
  2. Click New Project.
  3. Give your project a name — something like “My First Board.”
  4. Choose a board size (the default 830-point breadboard works great for learning).
  5. Click Create.

Your new, empty breadboard appears on the canvas — ready for components and wires.

5. Save your work

Press Ctrl+S (or ⌘+S on Mac) at any time to save. You’ll see a brief confirmation that your project has been saved.

If you try to navigate away with unsaved changes, NoWires.AI will warn you so you don’t lose anything.

💡 Tip: Save often! It only takes a second and keeps your work safe.

What you learned

  • How to sign up and access the app
  • The three main interface areas: sidebar, canvas, and toolbar
  • What each sidebar panel does
  • How to create a new project and save it
