The Editor

Master the canvas controls — zoom, pan, undo, keyboard shortcuts, and the difference between Design Mode and Run Mode.

The Editor

Now that you know your way around the interface, let’s get comfortable with the canvas — the space where all building happens.

What you’ll learn

  • How to zoom and pan around the board
  • Undo and redo actions
  • Essential keyboard shortcuts
  • The difference between Design Mode and Run Mode
  • How to toggle pin labels

1. Zoom

You have several ways to zoom in and out of the board:

  • Scroll wheel — scroll up to zoom in, down to zoom out
  • Keyboard — press + (or =) to zoom in, - to zoom out
  • Toolbar buttons — click the + and − buttons in the toolbar
  • Reset — press 0 to reset the view to the default zoom level

💡 Tip: Zoom is centered on your cursor position, so point at the area you want to focus on before scrolling.

2. Pan

To move around the board without zooming:

  • Click and drag on an empty area of the canvas to pan
  • This works in any tool mode (select, wire, erase)

When you’re zoomed in on a large board, panning is the fastest way to navigate between different sections of your circuit.

3. Undo and redo

Made a mistake? No problem:

  • Undo — press Ctrl+Z (or ⌘+Z on Mac)
  • Redo — press Ctrl+Shift+Z (or ⌘+Shift+Z on Mac)

Undo works for placing components, drawing wires, deleting, moving — pretty much any action on the board.

4. Keyboard shortcuts

These shortcuts keep your hands on the keyboard and your workflow fast:

KeyAction
VSelect tool — click to select components and wires
WWire tool — click two holes to draw a wire
EErase tool — click to remove wires and components
DeleteRemove the currently selected item
EscCancel the current action or close a panel
Ctrl+ZUndo
Ctrl+Shift+ZRedo
Ctrl+SSave
+ / =Zoom in
-Zoom out
0Reset view

💡 Tip: You can switch between tools without touching the toolbar. Press V, W, or E and start working immediately.

5. Design Mode vs Run Mode

The toolbar has a toggle that switches between two modes:

Design Mode (default)

  • Place, move, and delete components
  • Draw and remove wires
  • Edit project properties
  • The board is fully editable

Run Mode

  • The board becomes read-only — no editing allowed
  • Live measurements from connected hardware appear on the board
  • Voltage and current readings update in real time
  • Use this mode when your physical NoWires board is connected and powered

You’ll spend most of your time in Design Mode. Switch to Run Mode when you’re ready to test with real hardware. We’ll cover Run Mode in detail in the Run Mode & Live Measurements tutorial.

6. Pin labels

By default, component pins show labels (like VCC, GND, OUT). You can toggle these on or off:

  • Open Settings (gear icon in the sidebar)
  • Toggle Pin Labels on or off

Hiding pin labels gives you a cleaner view of the board. Showing them helps when you’re wiring up unfamiliar components.

What you learned

  • Multiple ways to zoom and pan the canvas
  • Undo/redo to fix mistakes instantly
  • Keyboard shortcuts for fast tool switching
  • Design Mode is for building, Run Mode is for measuring
  • Pin labels can be toggled for clarity
