Creating Custom Components

Design your own components with the Component Designer — choose a package, set pins, customize appearance, and save to your library.

Creating Custom Components

The built-in parts library covers common components, but you’ll eventually need something specific — a particular IC, a sensor module, or a custom breakout board. The Component Designer lets you create exactly what you need.

What you’ll learn

  • How to open the Component Designer
  • Choosing a package type
  • Setting pin count and dimensions
  • Customizing appearance
  • Naming pins
  • Saving to your library
  • Placing, editing, and deleting custom components

1. Open the Component Designer

There are two ways to start:

  • Create new: Open the Parts drawer → click the + button
  • Edit existing: Find a component in My Library → click the edit icon

Both open the Component Designer panel.

2. Choose a package type

The designer supports several common package styles:

PackageDescriptionExample
DIPDual Inline Package — pins on two sidesATmega328, 555 timer, op-amps
SIPSingle Inline Package — pins on one sidePin headers, resistor networks
AxialTwo-pin, inlineResistors, diodes
BoxRectangular body with configurable pinsModules, breakout boards

Select the package type that matches the real-world component you’re modeling.

3. Set pin count and body dimensions

After choosing a package:

  1. Set the number of pins — for DIP packages, this is evenly split between the two rows (e.g., 8 pins = 4 per side)
  2. Adjust body width and height if needed — the defaults are based on standard sizes for the chosen package type

The preview updates in real time as you make changes.

💡 Tip: Check the datasheet for your real component to get the correct pin count and package dimensions.

4. Customize body color

Click the color picker to change the component body color. This is purely visual but helps with identification on the board:

  • Black for ICs
  • Brown for resistors
  • Blue for capacitors
  • Green for custom modules
  • Or any color you prefer

5. Name each pin

This is the most important step for usability. Click on each pin to assign a name:

  1. Click a pin in the preview.
  2. Type the pin name (e.g., VCC, GND, OUT, IN1, CLK).
  3. Press Enter or click the next pin.

Pin names appear as labels on the board when pin labels are enabled, making wiring much easier.

💡 Tip: Use your component’s datasheet to match pin names exactly. This helps anyone reading your circuit (including future you).

6. Add metadata

Before saving, fill in the component details:

  • Name — what this component is called (e.g., “ATtiny85”, “DHT22 Sensor”)
  • Description — a brief note about what it does
  • Tags — keywords for searching (e.g., “microcontroller”, “temperature”, “sensor”)

7. Save to My Library

Click Save. Your component is now stored in the My Library section of the Parts drawer. It’s available in every project going forward.

8. Place your custom component

Using a custom component is identical to using a built-in one:

  1. Open the Parts drawer.
  2. Switch to the My Library tab.
  3. Find your component.
  4. Drag it onto the board.

It behaves exactly like a built-in part — you can move, rotate, flip, lock, and wire it.

9. Edit an existing custom component

Need to fix a pin name or add a pin?

  1. Open the Parts drawer.
  2. Go to My Library.
  3. Click the edit icon on the component you want to change.
  4. The Component Designer opens with the current configuration loaded.
  5. Make your changes.
  6. Click Save.

Changes apply to future placements. Components already on your boards keep their original configuration.

10. Delete a component from your library

  1. Open the Parts drawer.
  2. Go to My Library.
  3. Click the delete icon on the component.
  4. Confirm the deletion.

This removes the component from your library. Instances already placed on boards are not affected.

What you learned

  • Opening the Component Designer via the + button or edit icon
  • Choosing between DIP, SIP, Axial, and Box package types
  • Setting pin count and body dimensions
  • Customizing body color for identification
  • Naming pins from the datasheet
  • Adding name, description, and tags
  • Saving to My Library for use across projects
  • Editing and deleting custom components
