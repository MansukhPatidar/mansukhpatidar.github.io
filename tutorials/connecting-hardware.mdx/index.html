Connecting Your NoWires Board

The NoWires hardware board bridges the gap between virtual design and real circuits. Once connected, your software design controls real hardware — and real measurements flow back to your screen.

What you’ll learn

  • What the NoWires hardware board is
  • How to register a device with the challenge code
  • Pairing and unpairing devices
  • Understanding the connection status indicator
  • How cloud and LAN connections work
  • Troubleshooting common connection issues

1. What is the NoWires hardware board?

The NoWires board is a physical device that sits alongside your real breadboard. It:

  • Reads voltage and current measurements from your circuit
  • Communicates wirelessly with the NoWires.AI app
  • Has a small display showing its status and a challenge code for pairing
  • Connects over WiFi — no USB cables needed

Think of it as a smart multimeter that’s always connected to your breadboard.

2. Open the Boards drawer

Click the Boards icon in the sidebar to open the Boards drawer. This is your hub for managing all connected hardware devices.

If you haven’t registered any devices yet, you’ll see a prompt to add one.

3. Register a device

To link your physical board to your account:

  1. Power on your NoWires board.
  2. Look at the board’s display — it shows a challenge code (a short alphanumeric code).
  3. In the Boards drawer, click Register Device.
  4. Enter the challenge code from the display.
  5. Click Submit.

The board is now registered to your account and appears in the Boards drawer.

💡 Tip: The challenge code changes periodically for security. Enter it promptly after powering on the board.

4. Pair and unpair devices

Once registered, you can pair a device to your current project:

  • Pair — click the Use button next to the device in the Boards drawer. The device is now linked to your active project and will send/receive data.
  • Unpair — click the button again to disconnect the device from the current project.

You can register multiple devices but only pair one to a project at a time.

5. Connection status indicator

The Boards drawer shows a status dot next to each device:

IndicatorMeaning
🟢 Green dotConnected — the device is online and communicating
🔴 Red dotDisconnected — the device is offline or unreachable

The status updates in real time based on heartbeat signals from the device.

6. How connection works

Your NoWires board connects through two paths:

Cloud connection (Supabase Realtime)

  • Works from anywhere with internet
  • The board and the app both connect to the Supabase Realtime service
  • Messages are relayed through the cloud
  • Slightly higher latency but works across networks

Local connection (LAN WebSocket)

  • Works when the board and your computer are on the same WiFi network
  • Direct WebSocket connection for low latency
  • Faster measurement updates
  • Falls back to cloud if local connection isn’t available

The app automatically uses the best available connection.

7. Troubleshooting

Board won’t connect

  1. Check WiFi — make sure the board is connected to your WiFi network. If the board’s display shows a WiFi setup screen, it needs to be configured first.
  2. Same network — for LAN connections, ensure your computer and the board are on the same WiFi network.
  3. Power cycle — turn the board off and on again. It will reconnect automatically.

Connection drops frequently

  • WiFi signal — move the board closer to your router
  • Network congestion — the board uses minimal bandwidth, but a congested network can cause delays
  • Heartbeat — the board sends periodic heartbeats. If several are missed, the status changes to disconnected. It reconnects automatically when communication resumes.

Challenge code expired

Power cycle the board to get a fresh challenge code, then try registering again.

💡 Tip: The board reconnects automatically after brief network interruptions. If the status dot turns red, give it a few seconds before troubleshooting.
