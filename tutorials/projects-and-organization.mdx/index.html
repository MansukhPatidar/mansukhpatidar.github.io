Projects & Organization

As you build more circuits, you’ll want to keep them organized. The Project Organizer is your command center for managing all your work.

What you’ll learn

  • How to open the Project Organizer
  • Creating new projects with the wizard
  • Board variant selection
  • Editing project properties
  • Duplicating and deleting projects
  • Exporting a project as JSON
  • How unsaved changes protection works

1. Open the Project Organizer

Click the Projects icon in the sidebar. The Project Organizer slides open, showing a list of all your saved projects with their names, thumbnails, and last-modified dates.

2. Create a new project

  1. Click New Project at the top of the organizer.
  2. The New Project wizard opens.
  3. Enter a name for your project.
  4. Optionally add a description and tags to help you find it later.
  5. Choose a board variant (see below).
  6. Click Create.

Your new project opens immediately on the canvas, ready for building.

3. Board variant selection

When creating a project, you can choose from different breadboard sizes:

The default 830-point board is the most common and works for the majority of circuits. Larger boards give you more space for complex projects.

💡 Tip: If you’re not sure which size to pick, start with the default. You can always start a new project with a bigger board if you run out of space.

4. Project properties

To edit a project’s details after creation:

  1. Open the Project Organizer.
  2. Find your project in the list.
  3. Click the project to view its properties.
  4. Update the name, description, or tags as needed.

Good descriptions and tags make projects easier to find as your library grows.

5. Duplicate a project

Want to experiment without risking your working circuit?

  1. Open the Project Organizer.
  2. Find the project you want to copy.
  3. Click Duplicate.

A copy appears in your project list with “(Copy)” appended to the name. The duplicate is completely independent — changes to one don’t affect the other.

💡 Tip: Duplicate before making big changes. It’s a free safety net.

6. Delete a project

  1. Open the Project Organizer.
  2. Find the project you want to remove.
  3. Click Delete.
  4. Confirm the deletion when prompted.

Deleted projects cannot be recovered, so be sure before you confirm.

7. Export as JSON

To save a backup or share a project file:

  1. Open the project you want to export.
  2. Use Save As to export the project as a JSON file.
  3. The file downloads to your computer.

This JSON file contains everything — components, wires, positions, properties. You can share it with anyone or keep it as an offline backup.

8. Unsaved changes protection

NoWires.AI watches for unsaved changes. If you try to:

  • Close the browser tab
  • Navigate to a different page
  • Open a different project

…you’ll see a warning dialog asking if you want to save first. This prevents accidental data loss.

The toolbar shows an indicator when you have unsaved changes, so you always know your current save state.
