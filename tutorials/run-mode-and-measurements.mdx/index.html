Run Mode & Live Measurements

Run Mode is where your virtual design meets reality. When you switch from Design Mode to Run Mode, your board becomes a live dashboard — displaying real-time measurements from your connected hardware.

What you’ll learn

  • How to switch to Run Mode
  • What changes when Run Mode is active
  • Reading real-time voltage and current
  • Understanding the 5 measurement channels
  • Syncing your design to hardware
  • How the net mapper works

1. Switch to Run Mode

In the toolbar, find the Design/Run mode toggle. Click it to switch to Run Mode.

Before switching, make sure:

  • You have a NoWires board paired to your project (see Connecting Your NoWires Board)
  • The board status shows a green dot (connected)

2. What changes in Run Mode

When Run Mode is active:

FeatureDesign ModeRun Mode
Place components✅❌
Draw/delete wires✅❌
Move components✅❌
Live measurements❌✅
Board editing✅❌ (read-only)

The board becomes read-only to prevent accidental changes while you’re measuring. Your circuit layout is preserved exactly as you designed it.

💡 Tip: Design your circuit fully in Design Mode first, then switch to Run Mode to test. Switch back to Design Mode when you need to make changes.

3. Real-time voltage and current readings

Once in Run Mode with a connected board, you’ll see live measurements overlaid on your circuit:

  • Voltage — displayed in volts (V) at each measurement point
  • Current — displayed in milliamps (mA) where applicable

Readings update continuously as long as the board is connected and powered. If a reading changes (e.g., you adjust a potentiometer on your physical breadboard), you’ll see the update in real time on screen.

4. The 5 measurement channels

Your NoWires board has 5 measurement channels. Each channel can read voltage and current at a specific point in your circuit.

The channels are mapped to physical connections on your board. You can see which channel corresponds to which net in your circuit layout.

💡 Tip: For best accuracy, ensure your physical breadboard wiring matches your virtual design exactly. The measurements are only meaningful when the physical and virtual circuits align.

5. Syncing your design to hardware

Before measurements make sense, your virtual circuit needs to match your physical one. The sync process:

  1. Design your circuit in Design Mode.
  2. Build the same circuit on your physical breadboard.
  3. Connect and pair your NoWires board.
  4. Switch to Run Mode.

The app knows how your virtual components and wires map to physical breadboard positions, so it can display measurements in context on your virtual board.

6. How the net mapper works

Behind the scenes, the net mapper translates your virtual circuit into hardware connections:

  • It identifies which virtual nets correspond to which physical measurement points
  • It maps your component pins and wire connections to the board’s channel inputs
  • Measurement data from the hardware is routed to the correct virtual net for display

This all happens automatically — you don’t need to configure the mapping manually. Just make sure your physical wiring matches your virtual design.
